READ ME - Armed & Dangerous

Gameplay-
	Swing your Gladiator's arm to hit enemy Gladiators.
	Collect the Gold Coins that enemies drop or fall from the sky.

Button Layout-
	Character Movement 
	& Character Rotation - Left Joystick
	

	Swing Arm - Right Joystick
	
	Lift Arm - Right Trigger
	
	
Lower Arm -
 Left Trigger
	
	Click Buttons on Screen - Left Mouse Button
	
	
Pause Match -
 Controller Start Button / Left Mouse Button
	
	Exit Match End Screen - Any button / Left Mouse Button
